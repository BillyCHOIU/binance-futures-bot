# FluxBot · 合约智控

币安 **U 本位永续** 自动交易（Windows 点按钮）。

**决策链：舆情定方向 → 指标过滤进出场 → 硬风控保命**

> 不保证盈利。合约有爆仓风险。API **不要开提现权限**。

---

## 一分钟上手（开发模式）

1. 安装 [Python 3.11+](https://www.python.org/downloads/)（勾选 Add to PATH）
2. 双击 **`start.bat`**
3. 在窗口填入币安 API Key / Secret  
   - 建议先开 **Testnet**  
   - 实盘 API：只勾选「合约交易」，禁止提现
4. 点 **连接交易所** → **启动机器人**
5. 可点 **立即刷新舆情摘要** 看新闻方向

## 安装成桌面程序

见 **`INSTALL.md`**：`build.bat` 打包 → `install.bat` 安装到本机并生成桌面快捷方式。

配置文件：`config.yaml`（杠杆、币种、风控阈值等）  
密钥：`.env`（不要发给任何人）

---

## 功能（MVP Phase 0）

| 模块 | 说明 |
|------|------|
| 交易 | U 本位市价开平仓、杠杆、逐仓尝试 |
| 策略 | EMA 趋势 + ATR 止损/止盈 |
| 舆情 | 币安公告 + RSS 媒体 + Reddit 热帖 → BULL/BEAR/NEUTRAL |
| 叠加 | **舆情为主**：偏多只做多、偏空只做空；中性不开新仓 |
| 风控 | 单笔风险%、日亏熔断、回撤熔断、最大持仓数 |
| 界面 | 保存 Key / 连接 / 启停 / 全部平仓 / 日志 |
